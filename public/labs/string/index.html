/labs/string n°01

A shared room of cybernetic strings, ringing for one another.

Each visitor plays one string. A single pluck travels outward, moves nearby strings, and returns as sympathetic sound – this room is always live, always open.

It is a small reminder that everyday connection is real. Known or unknown, small actions ripple across other lives. If you're in this room and see another string, then you're not alone.
